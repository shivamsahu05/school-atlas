# API DOCUMENTATION - SAMS ATLAS PLATFORM

## Base Configuration
- **Base URL**: `/api`
- **Authentication**: Bearer Token (JWT) sent via `Authorization` header.
- **Content-Type**: `application/json` (except for `multipart/form-data` during file uploads).

## 1. Authentication & Identity
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/auth/login` | POST | Public | Authenticates user and returns JWT. Payload: `{username, password}` |
| `/auth/me` | GET | Any | Returns profile details for the currently logged-in token. |

## 2. Dashboard Analytics
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/dashboard/admin-metrics` | GET | Admin | Returns aggregate metrics (totals) for the Principal dashboard. |
| `/dashboard/teacher` | GET | Teacher | Returns customized metrics for a specific teacher's dashboard. |
| `/dashboard/homework-notifications` | GET | Any | Returns pending homework grading tasks. |
| `/dashboard/notifications` | GET | Any | Returns general unread alerts and system notifications. |

## 3. User Management
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/users` | GET, POST | Admin | Fetches or creates system users. |
| `/users/:id` | PUT, DELETE | Admin | Updates or deletes user profiles. |

## 4. Teacher Management
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/teachers` | GET, POST | Admin | Fetch all teachers or create a new teacher profile. |
| `/teachers/bulk-upload` | POST | Admin | Accepts a `.xlsx` file via `multipart/form-data` to bulk import teachers. |
| `/teachers/export` | GET | Admin | Downloads a blob file of teacher records. |
| `/teachers/:id/status` | PATCH | Admin | Blocks or unblocks a teacher account. |

## 5. Student Management
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/students` | GET, POST | Admin, Teacher | Fetch student list. |
| `/students/bulk-upload` | POST | Admin | Accepts Excel file to bulk enroll students. |
| `/students/:id/lifecycle` | POST | Admin | Upgrades, fails, or graduates a student. |
| `/students/:id/block` | PUT | Admin | Suspends a student account. |

## 6. Academic Setup (Classes, Sections, Subjects)
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/admin/classes` | GET, POST | Admin | Manage root class definitions. |
| `/admin/sections` | GET, POST | Admin | Manage section definitions. |
| `/admin/subjects` | GET, POST | Admin | Manage subjects definitions. |
| `/admin/subject-assignments`| POST | Admin | Assign subjects to a specific class. |
| `/classes` | GET | Any | Public fetch for class dropdowns. |

## 7. Syllabus & Micro-Schedule
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/syllabus` | GET, POST | Admin, Teacher | CRUD for syllabus tracking. |
| `/syllabus/plan` | GET | Any | Fetch structured weekly plan mapping. |
| `/syllabus/upload-syllabus` | POST | Admin | Bulk import syllabus via Excel. |
| `/teacher/schedule` | GET | Teacher | Fetches the current micro-schedule assigned to the logged-in teacher. |
| `/teacher/schedule/update` | POST | Teacher | Marks a specific micro-schedule topic as completed. |

## 8. Learning Outcomes (LO)
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/teacher-lo` | GET | Teacher | Fetches LO scores submitted by the teacher. |
| `/teacher-lo/self` | POST | Teacher | Submits LO evaluation for a student. |
| `/admin/lo` | GET | Admin | Retrieves all teacher LO submissions for review. |
| `/admin/lo/:id/score` | PUT | Admin | Principal awards final LO score, overriding or confirming the teacher's score. |

## 9. Observations
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/observations` | GET, POST | Admin | Principal submits a classroom observation audit form. |
| `/observations/teacher-own` | GET | Teacher | Teacher views their received observations. |

## 10. Performance & LMS Intelligence
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/performance/all` | GET | Admin | Retrieves pre-calculated overall KPI scores for all teachers. |
| `/performance/:teacherId/recalculate` | POST | Admin | Triggers the engine to instantly recalculate a teacher's KPI weights. |
| `/lms/intelligence/teacher-dashboard`| GET | Teacher | Returns AI/LMS generated insights for a teacher. |

## 11. Timetable
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/timetable/upload-excel` | POST | Admin | Imports master school timetable. |
| `/timetable/resolve-conflict`| POST | Admin | Fixes double-booking errors generated by the LMS validation engine. |

## 12. Leaves & Events
| Endpoint | Method | Role | Description |
|---|---|---|---|
| `/leave` | GET, POST | Any | Fetch or submit leave requests. |
| `/leave/:id` | PUT | Admin | Approve or Reject a leave request. |
| `/events` | GET, POST | Admin | Create or fetch school-wide events and notices. |
